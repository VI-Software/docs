# 📃 Select Server



## What is a server?

A server is a space created by a team of VI Software users, typically third parties to VI Software, who generate content on the platform. These users utilize the tools provided by VI Software to enhance the potential of their server and improve the experience for end users, which likely includes you.



## How can I select a server to view it in the launcher?

To select a server, follow these steps:

1. Log in at https://puf.visoftware.tech
2. In the top navigation bar, click on 'Servers.'

<figure><img src="../../.gitbook/assets/image.png" alt=""><figcaption></figcaption></figure>

3. Find the server you want to select. You can use the search bar (highlighted in red in the image below) to find it more quickly.

<figure><img src="../../.gitbook/assets/image (1).png" alt=""><figcaption><p>The search bar supports searches by server name, description, and Team</p></figcaption></figure>

4. Once you’ve found the server you're looking for, click the button that says 'View.'

<figure><img src="../../.gitbook/assets/image (2).png" alt=""><figcaption></figcaption></figure>

5. Scroll down to the controls section and click on 'Show Server.'

<figure><img src="../../.gitbook/assets/image (3).png" alt=""><figcaption></figcaption></figure>

{% hint style="danger" %}
**You need to restart the launcher to see the server** (if it was open before doing this)
{% endhint %}

6. Once you click 'Show Server,' your session will be redirected to the servers page. This means your request has been successfully completed, and you will now be able to see the server in the launcher.

_**Note:** To deselect a server, follow the same procedure. Instead of finding the 'Show Server' button, you will see one that says 'Hide Server' in the controls section._

{% hint style="info" %}
TIP for experience administrators: You can provide the server link directly to your users. The URL is static and will not change unless the server ID changes.
{% endhint %}
