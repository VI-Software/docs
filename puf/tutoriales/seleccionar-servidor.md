# 📃 Seleccionar Servidor



## ¿Qué es un servidor?

Un servidor es una experiencia desarrollada por un equipo de usuarios de VI Software, típicamente conformado por terceros a VI Software que generan contenido en la plataforma. Estos usuarios emplean las herramientas proporcionadas por VI Software para optimizar el potencial de su servidor y mejorar la experiencia de los usuarios finales, entre quienes probablemente se incluye usted.



## ¿Cómo puedo seleccionar un servidor para verlo en el launcher?

Para seleccionar un servidor, sigue los siguientes pasos:

1. Inicia sesión en[ https://puf.visoftware.tech](https://puf.visoftware.tech)
2. **En la barra de navegación superior**, haz clic en servidores.

<figure><img src="../../.gitbook/assets/image.png" alt=""><figcaption></figcaption></figure>

3. Busca el servidor que deseas seleccionar, puedes utilizar la barra de búsqueda (destacada en rojo en la imagen inferior) para encontrarlo más rápidamente.

<figure><img src="../../.gitbook/assets/image (1).png" alt=""><figcaption><p>La barra de búsqueda soporta búsquedas por nombre del servidor, descripción y Team</p></figcaption></figure>

4. Una vez hayas encontrado el servidor que buscas, haz clic en el botón que pone "Ver"

<figure><img src="../../.gitbook/assets/image (2).png" alt=""><figcaption></figcaption></figure>

5. Baja hacia la sección de controles y haz clic en "Mostrar servidor"

<figure><img src="../../.gitbook/assets/image (3).png" alt=""><figcaption></figcaption></figure>

{% hint style="danger" %}
**¡RECUERDA! Tienes que reiniciar el launcher para poder ver el servidor (si lo tenías abierto antes de hacer esto)**
{% endhint %}

6. Una vez haga clic en "Mostrar servidor" su sesión será redirigida a la página de servidores. Esto significa que se ha completado su solicitud con éxito, ahora podrá ver el servidor en el launcher.&#x20;

**Nota:** _Para deseleccionar un servidor, siga el mismo procedimiento. En lugar de encontrar el botón "Mostrar servidor", observará uno que indique "Ocultar servidor" en la sección de controles._

{% hint style="info" %}
TI**P para las administradoras de experiencias:** Puedes proporcionar el enlace del servidor directamente a tus usuarios, la URL es estática y no cambiará a no ser que cambie la ID de su servidor.
{% endhint %}
