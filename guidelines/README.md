# Guidelines

**Código de Conducta**: Establece normas de comportamiento para garantizar un ambiente inclusivo y respetuoso en la comunidad de desarrollo de VI.

**Términos de servicio gloables de VI Software:** Términos de servicio que son aplicados a todo servidor y proyecto de VI Software, ninguna otra normativa puede oponerse a estos términos de servicio.

**Directrices para Pull Requests (Solicitudes de Extracción)**: Define el proceso y las expectativas para proponer cambios al código existente, facilitando la colaboración entre desarrolladores y mantenedores del proyecto.

**Directrices para Contribuciones**: Detalla cómo contribuir al repositorio, incluyendo información sobre el formato de las contribuciones, la licencia de uso y otras consideraciones importantes.



Estas directrices son fundamentales para fomentar un desarrollo colaborativo y de alta calidad en el ámbito de VI Software, promoviendo estándares consistentes y un código robusto y confiable.
