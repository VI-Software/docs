# 📔 Política de Uso Justo para la API

## Política de Uso Justo para la API de Renderización de Skins de Minecraft de Vi Software

Última actualización: **10 de julio de 2024**

### 1. Introducción

La Política de Uso Justo tiene como objetivo garantizar que la API de Renderización de Skins de Minecraft de Vi Software sea utilizada de manera equitativa y eficiente por todos los usuarios. Al utilizar nuestra API, usted acepta cumplir con esta política, que complementa los términos de uso generales. Recuerda que, los términos de servicio de VI Software siguen vigentes.

### 2. Uso Permitido

Para garantizar un acceso justo y equitativo a todos los usuarios, se establecen los siguientes límites y prácticas recomendadas:

```
Límite de Solicitudes: Cada usuario puede realizar hasta 10,000 solicitudes a la API por día. Las solicitudes adicionales pueden estar sujetas a restricciones o tarifas adicionales.
Frecuencia de Solicitudes: No se permite realizar más de 5 solicitudes por segundo desde una misma dirección IP o clave de API.
Uso Eficiente: Utilice la API de manera que no genere una carga innecesaria en los servidores. Esto incluye la implementación de caché en su aplicación para evitar solicitudes repetitivas e innecesarias.
```

### 3. Prácticas No Permitidas

Para mantener la integridad y la eficiencia del servicio, las siguientes prácticas están prohibidas:

```
Abuso de la API: Cualquier uso que interfiera con el funcionamiento normal de la API o afecte negativamente la experiencia de otros usuarios. Esto incluye, pero no se limita a, ataques DDoS, scraping masivo o flooding.
Automatización Maliciosa: Utilizar scripts automatizados o bots para realizar un número excesivo de solicitudes o para eludir los límites establecidos.
Acceso No Autorizado: Intentar obtener acceso no autorizado a la API, sus servidores, sistemas o datos. Esto incluye la manipulación de encabezados de solicitud o el uso de credenciales robadas.
Uso Comercial No Autorizado: Utilizar la API para fines comerciales sin el permiso explícito por escrito de Vi Software.
```

### 4. Consecuencias del Incumplimiento

El incumplimiento de esta política puede resultar en medidas disciplinarias, que pueden incluir, pero no se limitan a:

```
Advertencias: Enviar una advertencia al usuario sobre el incumplimiento de la política.
Suspensión Temporal: Suspender temporalmente el acceso a la API.
Suspensión Permanente: Terminar el acceso a la API de manera permanente en casos de abuso grave o repetido.
Acciones Legales: Emprender acciones legales en caso de daños significativos o infracciones graves.
```

### 5. Excepciones y Contacto

Si su aplicación requiere un uso que excede los límites establecidos en esta política, puede solicitar una excepción poniéndose en contacto con nuestro equipo de soporte. Evaluaremos cada solicitud caso por caso y proporcionaremos una respuesta adecuada.&#x20;

### 6. Revisión y Modificación

Vi Software se reserva el derecho de revisar y modificar esta Política de Uso Justo en cualquier momento. Los cambios serán efectivos inmediatamente después de su publicación en nuestro sitio web o a través de la API. Su uso continuado de la API después de dichos cambios constituirá su aceptación de la política modificada.&#x20;
