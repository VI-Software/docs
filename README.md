# VI Software

## **Welcome to VI Software**

**Tools for Minecraft Modders, by Modders**

We’re a small team of Minecraft modders and server owners building open-source tools to make running and playing on modded servers easier.

**What We Offer**

| Tool                   | What It Does                                                                                                                                                              |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **VIS Launcher**       | Lets you connect to VI Software servers, manage mods, and switch accounts easily. No manual updates, no hassle.                                                           |
| **VIS Yggdrasil Auth** | Your server auth stays online even if Mojang’s auth is down. We only connect to Mojang once to verify your game purchase, after that, everything runs on our own servers. |
