# 📒 Política de Privacidad

Política de Privacidad de VIS Launcher Última actualización: 31 de julio de 2024

Bienvenido/a a VIS Launcher, una aplicación desarrollada por VI Software. En VI Software, valoramos y respetamos tu privacidad. Esta Política de Privacidad tiene como objetivo explicar cómo recopilamos, utilizamos y protegemos la información personal que nos confías a través de VIS Launcher.

### Definiciones

**"VIS Launcher"**: Se refiere a la aplicación desarrollada por VI Software para la gestión y uso de modpacks y servidores de juego.

**"VI Software"**: Empresa desarrolladora de VIS Launcher.

**"Datos de Inicio de Sesión"**: Información necesaria para acceder a VIS Launcher, incluyendo nombre de usuario y contraseña.

**"Datos Técnicos"**: Información recopilada sobre la configuración del dispositivo, dirección IP, tipo de dispositivo, versión del sistema operativo, y otros detalles técnicos del hardware y software del dispositivo.

**"API de VI Software"**: Interfaz de programación de aplicaciones utilizada por VIS Launcher para comunicarse con los servidores de VI Software.

**"Crash Logs"**: Registros de fallos del juego que incluyen información técnica sobre el rendimiento del dispositivo y la sesión de juego.

**"Peticiones HTTP a Servicios de Terceros"**: Solicitudes realizadas por VIS Launcher a servicios externos para obtener contenido, excluyendo datos personales salvo la dirección IP.

### Datos de Inicio de Sesión

1.1. **Recopilación de Información**: Al utilizar VIS Launcher, recopilamos la información necesaria para el inicio de sesión, que puede incluir tu nombre de usuario y contraseña.

1.2. **Conexiones con Servidores**: VIS Launcher establece conexiones con los servidores de VI Software para facilitar la descarga de modpacks, comprobar el estado de los servidores y verificar la autenticación.

### Datos Técnicos

2.1. **Recopilación Adicional**: VIS Launcher puede recopilar datos técnicos, como la configuración del dispositivo, la dirección IP, el tipo de dispositivo y la versión del sistema operativo. Estos datos se utilizan para mejorar la experiencia del usuario, garantizar la compatibilidad y aplicar medidas de seguridad, incluyendo el almacenamiento temporal de la dirección IP para gestionar ratelimits y medidas de seguridad en nuestra API.

2.2. **Envío de Datos a la API de VI Software**: Con el fin de garantizar la seguridad y mejorar nuestros servicios, VIS Launcher puede enviar datos técnicos, como el nombre de usuario, UUID de usuario, UUID de la instalación del sistema operativo, dirección IPv4 pública (o aplicable), hostname, servidor seleccionado e ID de Discord, a la API de VI Software. Algunos de estos datos pueden ser retenidos por un máximo de 1825 días naturales (\~ 5 años) por motivos de seguridad. Es importante destacar que estos datos nunca son compartidos con terceros y se utilizan únicamente para la identificación de bugs, mejorar la seguridad del sistema y garantizar la estabilidad de VIS Launcher.

2.3. **Envío de Crash Logs**: Además, es posible que VIS Launcher envíe crash logs del juego a la API de VI Software con el fin de identificar y solucionar problemas técnicos. Estos crash logs contienen información técnica sobre el fallo del juego y pueden incluir datos relacionados con el rendimiento del dispositivo y la sesión de juego.

2.4. **Peticiones HTTP a Servicios de Terceros**: VIS Launcher puede realizar peticiones HTTP a servicios de terceros, como Imgur, para obtener imágenes relacionadas con el contenido de la aplicación creado por terceros. Estas peticiones se realizan de manera segura y no incluyen datos personales del usuario (Excluyendo la IP, que es enviada en cualquier HTTP Request).

2.5. **Datos Adicionales para Seguridad y Estadísticas**: Para evitar evasiones de baneo y con fines de estadísticas técnicas, VIS Launcher también recopila información sobre la plataforma (ej: win32), tipo del dispositivo (ej: Windows NT), arquitectura del dispositivo (ej: x64), CPU del dispositivo y memoria RAM del dispositivo.

### Uso de la Información

3.1. **Propósito**: La información recopilada se utiliza exclusivamente para facilitar la funcionalidad de VIS Launcher, incluyendo la descarga de modpacks, la verificación del estado de los servidores de VI Software y la mejora continua de la aplicación.

3.2. **Confidencialidad**: Nos comprometemos a mantener la confidencialidad y la integridad de tus datos. Nunca compartiremos ni divulgaremos tu información con terceros sin tu consentimiento explícito, a menos que lo requiera la ley.

### Seguridad de la Información

4.1. **Medidas de Seguridad**: Hemos implementado medidas de seguridad adecuadas, incluyendo protocolos de cifrado y controles de acceso, para proteger la información recopilada.

### Leyes de Protección de Datos

5.1. **Cumplimiento Legal**: Esta política se ajusta a la Ley Orgánica 3/2018, de 5 de diciembre, de Protección de Datos Personales y garantía de los derechos digitales, así como a otras leyes y regulaciones aplicables en la Unión Europea y España.

### Cambios en la Política de Privacidad

6.1. **Actualizaciones Periódicas**: Esta Política de Privacidad puede actualizarse periódicamente para reflejar cambios en nuestras prácticas y servicios. Te recomendamos revisarla de manera regular para estar informado/a sobre cómo protegemos tu información.
