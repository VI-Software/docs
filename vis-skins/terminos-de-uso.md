La API de Renderización de Skins de Minecraft permite a los usuarios obtener y renderizar skins de Minecraft desde los servidores de autenticación de Vi Software. El acceso a esta API está sujeto a los términos y condiciones aquí descritos o en los [Términos de Servicio de VI Software](terminos-de-uso.md#terminos-de-uso-para-la-api-de-renderizacion-de-skins-de-minecraft).
